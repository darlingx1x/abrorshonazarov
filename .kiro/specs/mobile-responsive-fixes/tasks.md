## Notes

- Задачи помечены `*` являются опциональными и могут быть пропущены для быстрого MVP
- Каждая задача ссылается на конкретные требования для отслеживаемости
- Контрольные точки обеспечивают инкрементальную валидацию
- Property тесты валидируют универсальные свойства корректности
- Unit тесты валидируют конкретные примеры и граничные случаи
- Приоритет: исправление счетчиков → адаптация заголовка → оптимизация футера → улучшение взаимодействий
