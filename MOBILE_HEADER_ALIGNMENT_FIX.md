# Исправление выравнивания заголовка в мобильной версии

## Проблема

В мобильной версии заголовок с названием "Markdown Editor", "Вариант 17" и ФИО "Шоназаров Аброр • 085 23" выравнивался по центру, что приводило к наложению текста на кнопки справа.

## Решение

Изменено выравнивание заголовка только для мобильных устройств:

### Исправленные стили

```css
/* Для экранов менее 480px */
@media (max-width: 479px) {
  .header {
    padding: 0 var(--space-2);
    padding-right: 120px; /* Оставляем место для кнопок справа */
  }
  
  .header-content {
    text-align: left; /* Было: center */
  }
  
  .header-left {
    justify-self: start; /* Было: center */
  }
  
  .header-center {
    justify-self: start; /* Было: center */
  }
}
```

### Поведение по размерам экрана

- **< 360px**: Информация об авторе скрывается, остается только название и вариант
- **360px - 479px**: Все элементы выравниваются по левому краю
- **480px - 639px**: Стандартная трехколоночная компоновка (левый край | центр | правый край)
- **640px+**: Полная десктопная компоновка

### Результат

✅ Текст больше не накладывается на кнопки  
✅ Название "Markdown Editor" выравнивается по левому краю  
✅ "Вариант 17" остается рядом с названием  
✅ ФИО "Шоназаров Аброр • 085 23" выравнивается по левому краю  
✅ Кнопки остаются в правом верхнем углу  
✅ Достаточно места для всех элементов
